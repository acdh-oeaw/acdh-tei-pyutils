============
Installation
============


Stable release
--------------

To install acdh-tei-pyutils, run this command in your terminal:

.. code-block:: console

    $ pip install acdh-tei-pyutils

This is the preferred method to install acdh-tei-pyutils, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for acdh-tei-pyutils can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/acdh-oeaw/acdh-tei-pyutils

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/acdh-oeaw/acdh-tei-pyutils/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ make install


.. _Github repo: https://github.com/acdh-oeaw/acdh-tei-pyutils
.. _tarball: https://github.com/acdh-oeaw/acdh-tei-pyutils/tarball/master
